Neo4j Included Documentation
=========================================

Neo4j included documentation, providing reference material
and tutorials.

In addition to topical documents, subdirectories include:

* manual/text/neo4j-manual.txt - Neo4j Manual in pure text format
* manual/html/index.html - Neo4j Manual in navigable HTML format
* java/api/index.html - Javadocs for the Neo4j Libraries

See http://docs.neo4j.org/chunked/2.0.0/ for the Neo4j Manual in navigable HTML format (Online w comments).

For a PDF version of the Neo4j Manual, see http://docs.neo4j.org/
